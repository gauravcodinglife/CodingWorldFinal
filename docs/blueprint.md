# **App Name**: Cloud Dojo

## Core Features:

- Platform Visual Learning: Display AWS, GCP, Azure, Docker, and Kubernetes learning content visually.
- Interactive CLI Practice: Simulate a CLI environment where users can practice Docker and Kubernetes commands. Validate commands entered by user.
- Cloud Dashboard Simulations: Simulate cloud dashboards with live updating demo data. The AI tool will predict the resource requirements.
- User Authentication: Allow users to create accounts and log in to track their progress.
- Progress Tracking: Track user progress, including completed topics and CLI attempts.
- Content Management: Store and manage learning content, CLI commands, and dashboard metrics in Firestore.
- Personalized Recommendations: Leverage AI to suggest relevant learning paths and practice exercises based on user progress and preferences.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to evoke a sense of technology and trust.
- Background color: Dark gray (#1A202C) for a dark theme.
- Accent color: Electric purple (#701A75) for neon hover effects and highlights.
- Body text: 'Inter', a grotesque sans-serif font, will ensure readability and give a modern look. Headline Text: 'Poppins', a geometric sans-serif font, provides a contemporary and precise look for headings. Note: currently only Google Fonts are supported.
- Use minimalist icons that reflect cloud platforms and developer tools.
- Implement a responsive layout with a developer dashboard feel, glassmorphism cards, and a sticky navbar.
- Incorporate smooth hover effects and subtle transitions to enhance usability.
